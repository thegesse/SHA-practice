# RBGen (SHA-256 Rainbow Table Generator)

A lightweight, high-performance tool written in C for generating SHA-256 lookup tables from plaintext wordlists.

## Project Structure
- `main.c`: Handles file I/O and the user interface.
- `sha256/`: Contains the core cryptographic implementation.
- `Makefile`: Automates the build process.

## Prerequisites
- A C compiler (like `gcc`)
- `make` build utility
